Trocaire Fall Army Worm App

Produced an app as a member of a group for client, Trocaire, displaying information about the fall armyworm. The app could be used by farmers in developing countries to report fall armyworm infestations. This project was done using Android Studios, Java, SQL, and Python as part of the Programming Project Module at Trinity College Dublin.
